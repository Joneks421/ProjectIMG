Привествуем на

Первом сервере Minecraft для технического творчества при поддержке Федерального центра дополнительного образования

Особенности сервера

  1. Карта игровой зоны соответствует карте территорий Российской Федерации и включает все регионы Страны. Каждый игровой регион будет содержать определенный тип ресурсов, а участникам игрового сервера будет необходимо налаживать торговые отношения и выстраивать договоренности для взаимовыгодного сотрудничества
  2. Игровые модификации добавляют в игру новые технологические элементы и позволяют участникам находить новые типы ресурсов, необходимые для развития важных отраслей современной жизни (энергетика, космонавтика, биологические виды новых сельскохозяйственных культур и многое другое)
  3. Команда организаторов систематически проводит образовательные и развлекательные события (Ивенты): кейскарты, jumping пазлы, королевские битвы, командные головоломки и многое другое

В 2023 году сервер откроет свои двери для всех желающих, но уже сейчас самые преданные любители кубической песочницы могут подать заявку на beta-тестирование. Торопитесь!

Модификации

Galacticraft

Описание
Наверное самый популярный мод на ракеты и космос для майнкрафт, с его нововведениями можно будет построить многоуровневую ракету и отправиться в путешествие на другие планеты, изучать их, строить космические станции.

Industrial Craft 2

Описание
Большой и старый технический мод, который дарит игроку множество работающих на электричестве механизмов для самых разных сфер игровой жизни, включая добычу руд, обработку ресурсов и управление сельским хозяйством.

Farmer's Delight

Описание
Farmer's Delight создан для улучшения механик приготовления пищи и расширения фермерства в майнкрафт. Его цель добавит небольшое количество нового контента и расширить взаимодействие игрока с уже существующими предметами и блоками.

Button
MrCrayfish’s Furniture

Описание
Популярный мод на мебель в майнкрафт, который добавит большое количество разнообразной фурнитуры, например, появится: стол, стулья, диван, кресла, телевизор, компьютер и другие предметы. Все предметы доступны для крафта.

Button
Immersive Vehicles

Описание
Данная модификация - продолжение мода Flight Simulator, она добавит в майнкрафт реалистичные механики для огромного количества транспортных средств. Все средства передвижения разбиты на паки, необходимые для установки.

Create

Create предлагает множество инструментов и блоков для создания и декорации построек, а так же автоматизации. Новые элементы дадут игроку на выбор больше вариантов дизайна, обработка предметов будет происходить не в одном блоке.
